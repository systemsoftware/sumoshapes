PUBLISH A STAGE

cloud_uploadStep 1: Upload to GitHub

  1. Create a free GitHub account at github.com if you don't have one.
  2. Create a new repository on GitHub.
  3. Upload your stage files to the repository.
  4. Make sure to name the main file index.json.

Example repository structure:

my-sumo-stage/
├── index.json
└── README.md

local_offerStep 2: Add Topic

Add the topic "sumo-shapes-stage" to your repository. This identifies your repository as containing a stage for Sumo Shapes.

open_in_newStep 1: Open the tool

Click the button below to open the upload tool. If the project has been inactive for a while, it may take a few seconds to load.


loginStep 2: Sign in with GitHub

  1. Create a free GitHub account at github.com if you don't have one.

  2. Sign in with your GitHub account to access the upload tool.


cloud_uploadStep 3: Upload your stage

  1. Input your stage name into the text box.
  2. Describe your stage in the description box.
  3. Upload the stage files by clicking the "Choose a file" button.
  4. Click the button to publish your stage.

codeOptional: View the Code

If you wish to review the source of this tool, it is open source and available on Glitch.

check_circleYou're Done!

Congratulations! Your stage is now published and available for anyone to play directly from the Stage Selector.

Verify your stage:

  1. Input your username and stage name below.
  2. Click the "Verify" button.
  3. If your stage is found, you can play it in the Stage Selector!
